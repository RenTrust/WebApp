# RenTrust Web App
Full Stack
	Front end - AngularJS
	Back end - NodeJS
	UI/UX - CSS and Bootstrap
	Markup language - HTML5
	Scripting language - JavaScript
	Database - MongoDB
	Middleware - RabbitMQ
	SQL caching - Redis

Steps to run:

Start rabbitmq
Start mongodb
Start redis 
Start server
node www instead of double-click
